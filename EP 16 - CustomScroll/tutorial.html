مرحله ۱

شناسایی اسکرول بار

ابتدا باید pseudo-element اصلی اسکرول بار را تعریف کنید

::-webkit-scrollbar { width: 12px; }

این کد عرض اسکرول بار را 12 پیکسل تعیین می‌کند

مرحله ۲

طراحی مسیر اسکرول

حالا زمینه اسکرول بار (track) را استایل دهید

::-webkit-scrollbar-track { background: #f1f1f1; border-radius: 6px; }

این قسمت مسیری است که دکمه اسکرول روی آن حرکت می‌کند

مرحله ۳

رنگ‌آمیزی دکمه اسکرول

دکمه اسکرول (thumb) را با رنگ بنفش زیبا کنید

::-webkit-scrollbar-thumb { background: linear-gradient(45deg, #8a2be2, #9370db); border-radius: 6px; }

گرادیانت بنفش زیبایی به اسکرول بار می‌دهد

مرحله ۴

افکت هاور نهایی

وقتی ماوس روی اسکرول قرار گیرد، رنگ تیره‌تر شود

::-webkit-scrollbar-thumb:hover { background: linear-gradient(45deg, #7b1fa2, #8e24aa); }

🎉 تبریک! اسکرول بار شخصی‌سازی شده شما آماده است
